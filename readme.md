\## Описание

Данный продукт был разработан в рамках производственной практики. Программа позволяет пользователям иметь возможность контролировать рацион питания и соблюдать нормы КБЖУ, которые автоматически рассчитываются каждый день на основе параметров пользователя и списка съеденной еды.

<br>



\## Установка

Для установки программы нужно зайти в раздел `releases` и установить `.apk` из самого нового релиза, после чего найти скачанный файл в своём файловом менеджере и нажать на него. В открывшемся окне подтвердить установку <br>

После завершения процесса окно закроется и приложение будет установлено на устройство.

<br>



\## Использование

Данное приложение позволяет самостоятельно работать с рационом питания. Пользователь вводит дату рождения, рот, вес, пол и уровень активности. На основе этих данных рассчитывается КБЖУ на день. В приложении есть возможность составить базу из продуктов и выбирать их при добавлении съеденной еды с указанием граммовки, что автоматически определит количество КБЖУ. На основе списка съеденных за день продуктов рассчитывается количество КБЖУ и наглядно показывается на экране в виде полос заполнения этих 4 показателей. Предыдущие дни можно посмотреть в разделе "Общая статистика"
